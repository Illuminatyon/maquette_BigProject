# Maquette Big Project

## Description
A concise description of the project, explaining its purpose and functionalities. This project aims to provide users with effective solutions for design and prototyping.

## Setup Instructions
1. Clone the repository: `git clone https://github.com/Illuminatyon/maquette_BigProject.git`
2. Navigate into the project directory: `cd maquette_BigProject`
3. Install dependencies: `npm install`

## Basic Usage
To start the application, run the following command: `npm start`. This will launch the project and open it in your default web browser.
